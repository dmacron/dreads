# 📚 Dreads

A modern React-based web application that allows users to search, explore, and organize books into personalized collections. Built with Google Books API and Firebase, with aesthetic UI themes and smooth user experience.

🌐 **Live Demo:** https://dreads.vercel.app/

---

## 🚀 Features

- 🔍 Search books by title, author, or genre  
- 💾 Save books to your personal library  
- ❤️ Like books and manage "Want to Read" list  
- 🎨 Multiple UI themes (girly, pixel, modern, café)  
- 🔄 Real-time updates with Firebase  
- 📱 Fully responsive design  

---

## 🛠️ Tech Stack

- **Frontend:** React, Vite, CSS  
- **Backend/Database:** Firebase (Firestore + Authentication)  
- **API:** Google Books API  
- **Deployment:** Vercel
